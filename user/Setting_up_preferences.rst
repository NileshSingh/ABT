Setting up preferences
======================

Select ``Preferences`` option from master menu to edit organisation details and Add/Edit/Delete project.

Edit organisation details
-------------------------

  #. Displays previously saved organisation information, all the fields are editable.

  #. Press ``Save`` to save the changes.

  #. Press ``Reset`` to clear all the fields.

	.. image:: images/edit_org.png
	   :name: ABT main page
	   :align: center

Add/Edit/Delete project
-----------------------

  #. It displays list of all projects added before.

	.. image:: images/project_list.png
	   :name: ABT main page
	   :align: center

  #. ``Plus button`` (see fig.) will help to add new project/projects.

	.. image:: images/add_proj.png
	   :name: ABT main page
	   :align: center
	
  #. Select project name from the list to Edit/Delete.

     (Note: Project which is under transaction cannot be deleted.)
